# Nutanix v4 Go SDK Code Samples

For usage instructions, see the `README.md` file in each code sample directory.

## Requirements

To run the Go code samples, ensure the following requirements are met:

1. [Install Go](https://golang.org/doc/install)
2. Install the **github.com/tidwall/gjson** package:

   ```
   go get -u github.com/tidwall/gjson
   ```
